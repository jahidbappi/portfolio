# Jahid Bappi — Portfolio

Premium SaaS-style portfolio built with Next.js, Tailwind CSS, and Framer Motion.

**Live:** [jahidbappi.vercel.app](https://jahidbappi.vercel.app)

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Geist typography

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Connected to Vercel. Push to `main` to deploy automatically.
